### [React Router DOM](https://github.com/ReactTraining/react-router)
NavBar using  React Router
```
<nav>
  <ul>
    <li>
      <NavLink to="/home">Home</NavLink>
    </li>
    <li>
      <NavLink to="/about">About</NavLink>
    </li>
  </ul>
</nav>

```
